# 🦠 COVID-19 Data Analysis (SQL Project)

## 🔍 Project Overview
This project analyzes COVID-19 data using SQL to extract insights related to cases, deaths, and vaccination trends.

## 📊 Key Analysis
- Total cases and death percentage
- Country-wise infection rates
- Vaccination progress analysis
- Data joins and aggregations

## 🛠 Tools Used
- SQL Server (SSMS)
- Excel (Dataset)

## 📁 Dataset
- CovidDeaths.xlsx
- CovidVaccinations.xlsx

## 🎯 Conclusion
This project demonstrates SQL skills including joins, aggregations, and data analysis on real-world datasets.
